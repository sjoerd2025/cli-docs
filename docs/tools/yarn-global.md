# yarn-global

Popular packages installed globally via Yarn

## Serve

**Simple HTTP server for static files**

### Commands

| Command | Description |
|---------|-------------|
| `yarn global add serve` | Install globally |
| `serve` | Start server in current dir |
| `serve <dir>` | Serve directory |
| `serve -p 3000` | Custom port |
| `serve -l 5000 -s dist` | Single page app mode |
| `serve --cors` | Enable CORS |
| `serve --help` | Show help |
| `serve --version` | Show version |
| `serve --listen <addr>` | Custom address |
| `yarn global remove serve` | Uninstall globally |

## http-server

### Commands

| Command | Description |
|---------|-------------|
| `yarn global add http-server` | Install globally |
| `http-server` | Start server |
| `http-server -p 8080` | Custom port |
| `http-server -c-1` | Disable caching |
| `http-server --gzip` | Enable gzip |
| `http-server --cors` | Enable CORS |
| `http-server -o` | Open in browser |
| `http-server --version` | Show version |

## live-server

**Live reload static server**

### Commands

| Command | Description |
|---------|-------------|
| `yarn global add live-server` | Install globally |
| `live-server` | Start live server |
| `live-server --port=8000` | Custom port |
| `live-server --open=<path>` | Open specific path |
| `live-server --no-css-inject` | Disable CSS injection |
| `live-server --help` | Show help |
| `live-server --version` | Show version |
| `live-server --ignore=node_modules` | Ignore pattern |
| `live-server --wait=500` | Delay reload |
| `yarn global remove live-server` | Uninstall globally |
