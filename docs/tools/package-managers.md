## npm

**Node Package Manager for JavaScript package management**

### Commands

| Command | Description |
|---------|-------------|
| `npm install` | Install dependencies from package.json |
| `npm install <package>` | Install a specific package |
| `npm install -g <package>` | Install package globally |
| `npm update` | Update all dependencies to latest versions |
| `npm outdated` | Check for outdated dependencies |
| `npm audit` | Audit packages for security vulnerabilities |
| `npm run <script>` | Run a script defined in package.json |
| `npm list` | List installed dependencies |
| `npm search <keyword>` | Search npm registry for packages |
| `npm uninstall <package>` | Remove a package from project |

## yarn

**Fast, reliable, and secure dependency management for JavaScript**

### Commands

| Command | Description |
|---------|-------------|
| `yarn install` | Install dependencies from yarn.lock |
| `yarn add <package>` | Add a new dependency |
| `yarn add -D <package>` | Add a dev dependency |
| `yarn remove <package>` | Remove a dependency |
| `yarn upgrade` | Upgrade dependencies to latest versions |
| `yarn run <script>` | Run a script from package.json |
| `yarn audit` | Check for security vulnerabilities |
| `yarn cache clean` | Clear the yarn cache |
| `yarn why <package>` | Show why a package is installed |
| `yarn workspaces list` | List all workspaces in monorepo |
